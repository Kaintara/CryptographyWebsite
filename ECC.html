Elliptic Curve Cryptography

Elliptic Curves are used to create digital signatures that ensure authenticity in the ownership of cryptocurrencies like Bitcoin. This is due to a special property of Elliptic Curves that not only creates a harder version of the discrete logarithm problem but also generates significantly smaller private and public keys in practice.

What is an Elliptic Curve?

An Elliptic curve is a 2D curve which has the general equation y2 = x3 + ax + b . Elliptic curves usually look like the graph below but vary in shape based on the variables a and b.
